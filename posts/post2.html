JavaScript ES6 Features

JavaScript ES6 (ECMAScript 2015) introduced several powerful features that make coding more efficient and readable. Here are some of the key features:

1. Let and Const

The let and const keywords provide block-scoped variables, replacing the traditional var.

let name = "John";
const age = 30;

2. Arrow Functions

Arrow functions provide a concise syntax for writing functions.

const add = (a, b) => a + b;

3. Template Literals

Template literals allow embedding expressions in strings using backticks (`).

const greeting = `Hello, ${name}!`;

4. Default Parameters

Functions can have default parameter values.

function greet(name = "Guest") {
    return `Hello, ${name}!`;
}

5. Destructuring

Destructuring allows unpacking values from arrays or objects into variables.

const [a, b] = [1, 2];
const {name, age} = {name: "John", age: 30};

6. Modules

ES6 introduced modules for better code organization.

// Exporting
export const add = (a, b) => a + b;

// Importing
import { add } from './math.js';

These features, among others, have greatly improved the way we write JavaScript. Start using them to make your code cleaner and more efficient!
